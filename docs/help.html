UEFI Manager Help

General

UEFI Manager provides a Qt interface for inspecting and changing UEFI boot entries without dropping into command-line tools.

  • UEFI entries: review existing firmware entries, adjust boot order, set BootNext, toggle active entries, rename entries, and remove stale ones.
  • EFI stub install: prepare a direct-boot entry for a Linux installation by selecting the EFI System Partition and matching kernel options.
  • Frugal install: available on non-Arch builds for creating and managing frugal-style boot entries.

Notes

Privileged operations are routed through pkexec. Expect an authorization prompt only when a change requires elevated access.

Changes to boot order, active flags, BootNext, and EFI stub entries update firmware state. Review the selected entry carefully before applying changes on production systems.

On Arch builds, the Frugal tab is disabled by design.

Additional project documentation and support resources are available from the MX Linux documentation.
